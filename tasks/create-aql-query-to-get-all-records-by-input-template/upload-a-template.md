# Upload a template

To upload or create a template, we could use the following endpoint:

{% swagger method="post" path="/rest/openehr/v1/definition/template/adl1.4" baseUrl="https://ehr.portalkoding.com/ehrbase" summary="Upload a template" %}
{% swagger-description %}


[https://specifications.openehr.org/releases/ITS-REST/latest/definitions.html#definitions-adl-1.4-template-post](https://specifications.openehr.org/releases/ITS-REST/latest/definitions.html#definitions-adl-1.4-template-post)


{% endswagger-description %}

{% swagger-parameter in="header" name="Content-Type" type="String" required="true" %}
application/xml
{% endswagger-parameter %}

{% swagger-parameter in="body" name="template" type="String" required="true" %}
The XML representing the template
{% endswagger-parameter %}

{% swagger-response status="204: No Content" description="" %}
```javascript
{
    // Response
}
```
{% endswagger-response %}
{% endswagger %}
